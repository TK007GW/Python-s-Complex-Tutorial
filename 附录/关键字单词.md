# 关键字单词

## 如何查看python所有关键字

在Python交互环境中导入keyword包

```python
import keyword
print(keyword.kwlist)
```

## 关键字含义

| 关键字   | 含义                                           |
| -------- | ---------------------------------------------- |
| False    | 布尔类型的值,表示假                            |
| None     | none比较特殊,表示什么也没有,它有自己的数据类型 |
| True     | 布尔类型的值,表示真                            |
| and      | 用于表达式计算,逻辑与操作                      |
| as       | 用于类型转换                                   |
| assert   | 断言,用于判断变量的值是否为真                  |
| break    | 中断循环语句的执行                             |
| class    | 用于定义类                                     |
| continue | 跳出本次循环.继续执行下一次循环                |
| def      | 用于定义函数或方法                             |
| del      | 删除变量或序列的值                             |
| elif     | 条件语句,表示 否则如果                         |
| else     | 条件语句,表示否则                              |
| except            | except包含捕获异常后的操作代码块,也可用于异常和循环语句 |
| finally  | 用于异常语句,出现异常后,始终要执行finally包含的代码块      |
| for      | for循环语句 |
| from     | 用于导入模块,与Import始终使用 |
| global   | 定义全局变量 |
| if       | 条件语句,与else,elif结合使用 |
| import   | 用于导入模块,from结合使用 |
| in       | 判断变量是否在序列中 |
| is       | 判断变量是否是某个类的实例 |
| lambda   | 定义匿名函数 |
| nonlocal | 用于表示外部作用域的变量 |
| not      | 用于表达式运算,逻辑非运算 |
| or       | 用于表达式计算,逻辑或操作 |
| pass     | 空的类,方法或函数的站位类 |
| raise    | 异常抛出操作 |
| return   | 用于从函数返回计算结果 |
| try      | try包含可能出现异常的语句 |
| while    | while循环语句 |
| with     | 简化python的语句 |
| yield    | 用于从函数依次返回值 |
